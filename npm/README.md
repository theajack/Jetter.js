## jetterjs

dom操作和工具类js库

Js library for form operation and validation

[API](https://theajack.gitee.io/jetterjs)

### 1. npm 

```
npm i jetterjs
```

```js
import J from 'jetterjs'
```

### 2. cdn 

```html
<script src="https://cdn.jsdelivr.net/npm/jetterjs/jetter.min.js"></script>
<script>
    J.ready(function(){

    });
</script>
```
